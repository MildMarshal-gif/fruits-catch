# 変更事項整理（形式化）

## 1. Fever Time中の背景（丘）
- 変更対象: Fever中に描画される丘レイヤー
- 要件: 形状デザインは維持し、色のみ暗くする（明度を下げる）

## 2. Fever中の点数獲得表示
- 変更対象: フルーツ取得時に表示される点数ポップアップ
- 要件: `docs/attachments/fever-score-reference.png` を参考にした見た目へ変更

## 3. ヒット演出の色味
- 変更対象: 当たり時の演出カラー
- 要件:
  - フルーツ: 種類ごとに色味を変える
  - 虫/星: 色味調整をせず、素材そのままの色を使う

# 実装可否の振り分け

## すぐに実装できるもの
1. Fever中の丘の色を暗くする
   - 実装可
   - 補足: 背景描画処理内で、Fever強度に応じて丘カラーの明度を下げれば対応可能

2. フルーツごとのヒット演出色（虫/星は素材色維持）
   - 実装可
   - 補足: 現行実装でもほぼ要件を満たしており、最終調整のみで反映可能

## すぐに実装できないもの（要件確定が必要）
1. 点数表示を参考画像どおりにする件
   - 理由: 「どこまで忠実に再現するか」が未確定
   - 参考画像には、雲形プレート・虹グロー・`×2`バッジ・太字アウトラインなど複数要素が含まれるため、再現範囲の合意が必要

# ユーザーが決めるべきこと（未確定項目の解消）

## 推奨（おすすめ）
1. 再現レベルを選ぶ
   - A: 雰囲気再現（文字 + グロー + `×2`バッジ、プレート簡略）
   - B: 高忠実再現（プレート形状含めてほぼ同等）

2. 表示適用範囲を決める
   - Fever中のみ新デザインにする
   - 通常時にも同系デザインを適用する

3. 表示文言を決める
   - 現状の `base×2 = +got` を維持する
   - 数値中心表示（例: `30` + `×2`）へ変更する

4. 演出強度を決める
   - グロー強め / 標準 / 弱め

## 代案
1. 先に「背景の丘」「ヒット演出色」を実装し、点数表示はA案（雰囲気再現）で暫定導入
2. 暫定導入後にプレイ確認して、必要ならB案（高忠実再現）へ拡張
