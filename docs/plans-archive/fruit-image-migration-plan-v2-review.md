## 2. Findings（重大度順）

### 軽微-1: `spriteMeta` の初期値は実画像の実測で最終確定した方が安全
- 指摘:
  - `collisionRadiusPx` は暫定値のため、実装時に `naturalWidth/Height` とプレイ検証で最終チューニングが必要。
- 影響:
  - 実装後の微調整工数が発生する可能性がある。
- 対応:
  - 実装タスク内でチューニング前提を明示済み（Phase 6）。
